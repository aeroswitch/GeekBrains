1. Общее описание базы данных Spotify

	База создана для хранения данных музыкальных сервисов наподобие Spotify - https://www.spotify.com/ru-ru/ 

2. Какие задачи решает база данных и общее описание таблиц

	База хранит информацию и позволяет другим программам обращатся к следующим данным:
	- зарегистрированные пользователях и их данные (таблица users)
	- доступные типы подписок (тарифов) сервиса (таблица subscription_type),
	- историю заказов подписок всеми пользователями (таблица users_subscriptions_orders_history) и текущие подписки ползователя (таблица users_subscriptions)
	- артисты и их данные (таблица artists)
	- треки и релизы (таблицы songs, releases, songs_in_releases)
	- плейлисты созданные пользователями и публичные плейлейсты, созданные сервисом (таблицы user_playlists, public_playlists, songs_in_user_playlists, songs_in_public_playlists)
	- список треков, которые лайкнул пользователь (таблица songs_liked_by_users) и история прослушиваний треков (таблица users_songs_hisory)
	- миксы, составленные сервисом на основании лайков для конкретного пользователя (таблицы recommended_mixes_for_user, songs_in_mixes
	- общее объединение публичных плейлистов на основании выбранного жанра или настроения пользователем (таблицы genres_moods_categories, playlists_in_genres_moods_categories), а так же чарты самых популярных треков на основании кол-ва ежедневных прослушиваний (таблицы top_charts, songs_in_top_charts)
	- список артистов, плейлистов и чартов, которые фолловит пользователь (таблицы user_follows_artists, user_follows_public_playlists, user_follows_top_charts)
